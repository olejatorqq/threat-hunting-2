Описание

Вы – специалист по информационной безопасности компании “СуперМегатек”. Вы часто используете информацию о сетевом трафике для обнаружения подозрительной и вредоносной активности. Помогите защитить Вашу компанию от международной хакерской группировки AnonMasons.

Задание 1: Надите утечку данных из Вашей сети

Описание задания

Важнейшие документы с результатми нашей исследовательской деятельности в области создания вакцин скачиваются в виде больших заархивированных дампов. Один из хостов в нашей сети используется для пересылки этой информации – он пересылает гораздо больше информации на внешние ресурсы в Интернете, чем остальные компьютеры нашей сети. Определите его IP-адрес.

Метод решения

Нахождение строки src с максимальным значением bytes среди строк, которые начинаются с “12.”, “13.” или “14.” и dst у которых не начинается с “12.”, “13.” или “14.”

dataset %>%
  filter(str_detect(src, "^((12|13|14)\\.)")) %>%
  filter(!str_detect(dst, "^((12|13|14)\\.)")) %>%
  select(src, bytes) %>%
  group_by(src) %>%
  summarise(bytes = sum(bytes)) %>%
  arrange(desc(bytes)) %>%
  slice_head(n = 1) %>%
  select(src) 

# A tibble: 1 × 1
  src         
  <chr>       
1 13.37.84.125

Задание 2: Надите утечку данных 2

Описание задания

Другой атакующий установил автоматическую задачу в системном планировщике cron для экспорта содержимого внутренней wiki системы. Эта система генерирует большое количество траффика в нерабочие часы, больше чем остальные хосты. Определите IP этой системы. Известно, что ее IP адрес отличается от нарушителя из предыдущей задачи.

Метод решения

Для решения данной задачи необходимо понять, какое время является нерабочим. После этого необходимо убрать из возможных результатов, результат первого задания. Затем происходит группировка данных по уникальным значениям столбца src с вычислением суммы столбца bytes, для которых удовлетворяет условие выборки часов. После данные сортируются в порядке убывания суммы bytes и выводится первая строка данных, содержащая наибольшее значение суммы bytes.

График распределения количества пакетов за каждый час

dataset %>%
    select(timestamp, src, dst, bytes) %>%
    collect() %>%
    mutate(
        outside_traffic = (str_detect(src, "^((12|13|14)\\.)") & !str_detect(dst, "^((12|13|14)\\.)")),
        hour = hour(as.POSIXlt(timestamp/1000, origin = "1970-01-01"))
    ) %>%
    filter(
        src != "13.37.84.125",
        outside_traffic == TRUE,
        hour >= 0 & hour <= 15
    ) %>%
    group_by(src) %>%
    summarise(total_bytes = sum(bytes)) %>%
    arrange(desc(total_bytes)) %>%
    head(1)

Задание 3: Надите утечку данных 3

Описание задания

Еще один нарушитель собирает содержимое электронной почты и отправляет в Интернет используя порт, который обычно используется для другого типа трафика. Атакующий пересылает большое количество информации используя этот порт, которое нехарактерно для других хостов, использующих этот номер порта. Определите IP этой системы. Известно, что ее IP адрес отличается от нарушителей из предыдущих задач.

Метод решения

Для решения задачи необходимо сделать выборку данных согласно условиям (ip нарушителя отличается от ip нарушителей из предыдущих задач). После необходимо сортировать таблицу по порту и сгруппировать по источнику и порту. После этого происходит вычисление суммы и среднего значения байтов по порту, затем следует объединение данных. Если разница наименьшая среди отправленых байт по порту и среднего значения, то можно полагать, что ip с этой наименьшей разницей и является нарушителем.

График отправленных байтов по портам

Фильтрация и выбор данных по условиям

filtered_table <- dataset %>%
  filter(src != "13.37.84.125", src != "13.48.72.30", str_detect(src, "^((12|13|14)\\.)") & !str_detect(dst, "^((12|13|14)\\.)")) %>%
  select(src, bytes, port) %>%

Сортировка таблицы по порту и группировка по источнику и порту

sorted_table <- filtered_table[order(filtered_table$port, decreasing = TRUE), ] %>% 
                 group_by(src, port)

Вычисление суммы и среднего значения байтов по порту и объединение данных

total_bytes_table <- sorted_table %>% 
              summarize(port_sum = sum(bytes)) %>% 
              group_by(port)

average_bytes_table <- total_bytes_table %>% 
               summarize(port_mean = mean(port_sum))

merged_data_table <- merge(sorted_table, average_bytes_table, by = "port")

Выбор наименьшей разницы источника данных после фильтрации

filtered_data_ip <- filter(merged_data_table, 
                  str_detect(src, "13.37.84.125", negate = TRUE) & str_detect(src, "13.48.72.30", negate = TRUE)
                )
filtered_data_ip$difference <- filtered_data_ip$bytes - filtered_data_ip$port_mean
min_diff_index <- which.min(filtered_data_ip$difference)

min_diff_row <- filtered_data_ip[min_diff_index,]
